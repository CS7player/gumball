# Gumball

Gumball is a Visual Studio Code extension that lets you save and run frequently used commands in specific directories using a simple UI.

## Features

- Add commands with label, command, and path
- Run commands in a VS Code terminal
- Edit or delete saved commands
- Clean Tailwind CSSâ€“based UI

## Usage

1. Open Command Palette (`Ctrl + Shift + P`)
2. Run **Open Gumball**
3. Add your command, path, and label
4. Click **Run** to execute it in a terminal

## Requirements

- Visual Studio Code 1.107 or higher
